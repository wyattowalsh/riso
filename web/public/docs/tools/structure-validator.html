Structure Validator¶

The Structure Validator ensures project layout configurations are valid and consistent.

Purpose¶

  • Validates single-package vs monorepo layouts

  • Checks directory structure requirements

  • Ensures proper configuration file placement

  • Validates naming conventions

Location¶

template/hooks/validators/structure_validator.py

Validated Layouts¶

  • single-package: Standard Python package structure

  • monorepo: Multi-package workspace with apps/ and packages/

Usage¶

Automatically invoked during template generation to validate structure configurations.
