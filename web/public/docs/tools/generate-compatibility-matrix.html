Generate Compatibility Matrix¶

This script parses copier.yml to extract prompts, dependencies, and incompatibilities, then generates comprehensive compatibility documentation.

Purpose¶

  • Extract prompt definitions from copier.yml

  • Identify incompatible configuration combinations

  • Generate Mermaid decision tree diagrams

  • Document alternative recommendations and best practices

Location¶

scripts/generate_compatibility_matrix.py

Usage¶

Generate Documentation¶

uv run python scripts/generate_compatibility_matrix.py

Custom Output Path¶

uv run python scripts/generate_compatibility_matrix.py \
  --output docs/my-compatibility.md

With Dependency Visualization¶

uv run python scripts/generate_compatibility_matrix.py \
  --visualize-deps

  • Frontend framework compatibility (meta-frameworks, state management, components)
