## Hook Execution Sequence

```mermaid
sequenceDiagram
    autonumber
    participant U as User
    participant C as Copier
    participant Pre as pre_gen_project.py
    participant V as validators/
    participant T as Template Engine
    participant Post as post_gen_project.py
    participant O as Output Project

    U->>C: copier copy . project/
    C->>Pre: Execute pre-generation hook

    rect rgb(204, 251, 241)
        Note over Pre,V: Validation Phase
        Pre->>V: Check Python version
        V-->>Pre: ✓ 3.11+
        Pre->>V: Check uv installed
        V-->>Pre: ✓ Available
        Pre->>V: Check Node.js (if needed)
        V-->>Pre: ✓ 20 LTS
        Pre->>V: Check pnpm (if needed)
        V-->>Pre: ✓ ≥8
    end

    Pre-->>C: Validation passed

    rect rgb(219, 234, 254)
        Note over C,T: Rendering Phase
        C->>T: Load copier.yml
        C->>T: Process .jinja templates
        T->>O: Write rendered files
    end

    C->>Post: Execute post-generation hook

    rect rgb(254, 243, 199)
        Note over Post,O: Setup Phase
        Post->>O: Initialize .git
        Post->>O: Install pre-commit hooks
        Post->>O: Run uv sync
        Post->>O: Run pnpm install (if Node)
        Post->>O: Write .riso/metadata.json
        Post->>O: Run smoke tests
    end

    Post-->>C: Setup complete
    C-->>U: Project ready!
```

```mermaid
classDiagram
    class CopierHook {
        <<abstract>>
        +execute() bool
        +validate() ValidationResult

    class PreGenHook {
        +validate_tooling() bool
        +check_prerequisites() list
        +abort_if_missing(tools) void
```
