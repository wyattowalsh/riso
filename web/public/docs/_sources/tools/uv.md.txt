# uv

uv is an extremely fast Python package and project manager, written in Rust. It's a drop-in replacement for pip, pip-tools, and virtualenv.

## Features

- **Speed**: 10-100x faster than pip
- **Unified**: Package management, virtual environments, Python versions
- **Compatible**: Works with existing pyproject.toml
- **Lockfiles**: Reproducible builds with uv.lock

## Usage in Riso

uv is the default package manager:

```bash
uv sync                     # Install from lockfile
uv add <package>            # Add a dependency
uv remove <package>         # Remove a dependency
uv run <command>            # Run in virtual environment
uv lock                     # Update lockfile
```

## Dependency Groups

Riso uses uv's dependency groups:

```toml
[dependency-groups]
dev = ["pytest", "ruff"]
docs = ["sphinx", "shibuya"]
```

```bash
uv sync --group dev --group docs
```
