# Language Validator

The Language Validator ensures language version requirements are met and compatible.

## Purpose

- Validates Python version requirements (3.11+)
- Checks Node.js version compatibility
- Ensures tooling supports selected versions
- Validates type annotation compatibility

## Location

```
template/hooks/validators/language_validator.py
```

## Validated Versions

| Language | Minimum | Recommended |
|----------|---------|-------------|
| Python | 3.11 | 3.12+ |
| Node.js | 18 | 20+ |

## Usage

Validates that selected features are compatible with specified language versions.
