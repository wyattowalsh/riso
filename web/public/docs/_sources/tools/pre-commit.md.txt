# pre-commit

pre-commit is a framework for managing and maintaining multi-language pre-commit hooks. It ensures code quality checks run before every commit.

## Features

- **Multi-language**: Supports Python, JavaScript, Go, Rust, and more
- **Automatic**: Runs automatically on git commit
- **Fast**: Only runs on changed files
- **Ecosystem**: Large collection of existing hooks

## Usage in Riso

Install and run hooks:

```bash
uv run pre-commit install           # Install hooks
uv run pre-commit run --all-files   # Run manually
uv run pre-commit autoupdate        # Update hooks
```

## Configuration

Riso includes a `.pre-commit-config.yaml`:

```yaml
repos:
  - repo: https://github.com/astral-sh/ruff-pre-commit
    rev: v0.14.0
    hooks:
      - id: ruff
        args: [--fix]
      - id: ruff-format
```
