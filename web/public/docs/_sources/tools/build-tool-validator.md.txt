# Build Tool Validator

The Build Tool Validator ensures build tools and bundlers are properly configured for the selected stack.

## Purpose

- Validates frontend build tool selections
- Checks bundler compatibility with frameworks
- Ensures proper TypeScript configuration
- Validates build output configurations

## Location

```
template/hooks/validators/build_tool_validator.py
```

## Validated Build Tools

- **Vite**: Default for React, Vue, Svelte
- **Turbopack**: Next.js 15+ default
- **esbuild**: For pure TypeScript projects

## Usage

Automatically invoked when frontend options are enabled.
