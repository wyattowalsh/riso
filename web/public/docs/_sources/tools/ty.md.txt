# ty

ty is Astral's new Python type checker, designed for speed and compatibility. Built in Rust, it's 10-60x faster than mypy while maintaining broad compatibility.

## Features

- **Speed**: 10-60x faster than mypy
- **Compatibility**: Works with existing type annotations
- **Incremental**: Fast re-checking on file changes
- **Clear errors**: Human-readable error messages

## Usage in Riso

ty is the default type checker in Riso projects:

```bash
uv run ty check src/          # Check a directory
uv run ty check src/main.py   # Check a file
```

## Configuration

Riso configures ty in `pyproject.toml`:

```toml
[tool.ty.environment]
python-version = "3.11"

[tool.ty.src]
respect-ignore-files = true

[tool.ty.terminal]
output-format = "full"
```
