# WebSockets Module

The WebSocket module provides real-time bidirectional communication capabilities for your Riso application.

## Features

- **Real-time**: Bidirectional communication
- **Authentication**: JWT-based auth support
- **Scalable**: Redis pub/sub for multi-instance
- **Type-safe**: Pydantic message validation

## Usage in Riso

Enable `websocket_module: enabled` with `api_tracks: python`:

```bash
# WebSocket endpoint available at /ws
uv run uvicorn src.api.main:app --reload
```

## Generated Structure

```
src/
├── api/
│   └── websocket/
│       ├── __init__.py
│       ├── handlers.py   # Connection handlers
│       ├── events.py     # Event definitions
│       └── manager.py    # Connection manager
```

## Example

```python
from fastapi import WebSocket

async def websocket_endpoint(websocket: WebSocket):
    await websocket.accept()
    while True:
        data = await websocket.receive_text()
        await websocket.send_text(f"Echo: {data}")
```
