# Tool Validator

The Tool Validator ensures selected development tools are compatible and properly configured.

## Purpose

- Validates quality tool combinations
- Checks package manager compatibility
- Ensures CI/CD tool configuration
- Prevents conflicting tool selections

## Location

```
template/hooks/validators/tool_validator.py
```

## Validated Tools

- **Quality**: Ruff, ty, Pylint, pytest
- **Package Managers**: uv, pnpm
- **CI/CD**: GitHub Actions

## Usage

Automatically invoked during template generation to validate tool configurations.
