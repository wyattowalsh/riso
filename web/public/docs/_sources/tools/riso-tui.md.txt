# Riso TUI

The Riso TUI provides an interactive terminal user interface for configuring and generating projects using Textual.

## Purpose

- Interactive project configuration with real-time preview
- Conditional prompt visibility based on selections
- Live configuration summary with validation
- Keyboard-driven workflow for power users

## Location

```
src/riso/tui.py
```

## Usage

### Via Makefile

```bash
make tui
```

### Via CLI

```bash
uv run riso tui --dest ./my-project
```

### With Pre-configured Answers

```bash
uv run riso tui --dest ./my-project --answers ./answers.yml
```

## Features

### Real-time Configuration

- Live summary panel shows current selections
- Conditional prompts appear/hide based on context
- Destination path validation with overwrite warnings

### Keyboard Bindings

| Key | Action |
|-----|--------|
| `Tab` / `Shift+Tab` | Navigate between fields |
| `g` | Generate project |
| `q` | Quit without generating |

### Configuration Options

The TUI dynamically renders prompts from `template/copier.yml`:

- Project name and destination
- API tracks (Python, Node, both, or none)
- Frontend framework selection
- Documentation site configuration
- Optional modules (CLI, MCP, shared logic)

## Architecture

The TUI uses Textual's reactive architecture:

```python
class RisoTuiApp(App[Optional[TuiResult]]):
    def compose(self) -> ComposeResult:
        yield Header()
        yield Horizontal(prompt_column, summary_column)
        yield Footer()
```

### Components

- **PromptRow**: Renders individual prompts with labels and inputs
- **SummaryCard**: Live preview of configuration state
- **InputField/SelectField/MultiSelectField**: Form widgets
