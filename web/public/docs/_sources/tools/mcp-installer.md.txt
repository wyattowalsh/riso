# mcp-installer

```{rubric} Universal MCP Server Manager
```

```{admonition} Overview
:class: tip

AI agent skill for researching, installing, syncing, and managing MCP (Model Context Protocol) servers across all AI coding assistants including Claude Desktop, Claude Code, Cursor, GitHub Copilot, Gemini CLI, Amazon Q, and more.
```

## Usage

### Research Servers

```bash
# Search registries
uv run python -m scripts.research --query "github" --cwd .claude/skills/mcp-installer

# Batch search
uv run python -m scripts.research --batch "github,slack,notion" --cwd .claude/skills/mcp-installer

# Get specific server
uv run python -m scripts.research --server-id "modelcontextprotocol/server-github" --cwd .claude/skills/mcp-installer
```

### Install Servers

```bash
# Install to Claude Desktop + Code
uv run python -m scripts.cli install github --cwd .claude/skills/mcp-installer

# Install to specific interface
uv run python -m scripts.cli install slack --interface cursor --cwd .claude/skills/mcp-installer

# Install with custom config
uv run python -m scripts.cli install --config '{"name": "custom", "command": "npx", "args": ["-y", "my-server"]}' --cwd .claude/skills/mcp-installer
```

### Sync Across Interfaces

```bash
# Sync from Claude Code to ALL interfaces
uv run python -m scripts.sync --from claude-code --to all --cwd .claude/skills/mcp-installer

# Sync specific servers
uv run python -m scripts.sync --from claude-desktop --to cursor,windsurf --servers "github,slack" --cwd .claude/skills/mcp-installer

# Install and sync to all
uv run python -m scripts.sync --add "github,slack,notion" --to all --cwd .claude/skills/mcp-installer
```

### Validate Configuration

```bash
# Validate Claude Desktop config
uv run python -m scripts.validate --interface claude-desktop --cwd .claude/skills/mcp-installer

# Validate with auto-fix
uv run python -m scripts.validate --interface claude-code --fix --cwd .claude/skills/mcp-installer
```

### Check Status

```bash
# Show all interface statuses
uv run python -m scripts.sync --status --cwd .claude/skills/mcp-installer

# List installed servers
uv run python -m scripts.cli list --interface claude-code --cwd .claude/skills/mcp-installer
```

## Script Reference

| Script | Purpose |
|--------|---------|
| `scripts/research.py` | Multi-registry parallel search |

| `scripts/config.py` | Safe config read/write operations |

| `scripts/models.py` | Pydantic data models |
| `scripts/document.py` | Documentation registry management |

## Workflow

1. **Research** — Query registries (Official MCP Registry, Smithery)
2. **Install** — Add server to target interfaces with proper config format
3. **Sync** — Propagate across all configured AI tools
4. **Validate** — Verify JSON syntax and schema compliance

## Security Features

- **Credential Sanitization** — API keys redacted in logs
- **Atomic Writes** — Prevents config corruption
- **File Locking** — Prevents concurrent modification
- **Backup Creation** — Automatic `.bak` before changes
- **Permission Checks** — Warns on overly permissive configs

## Error Handling

| Error | Resolution |
|-------|------------|
| Config not found | Create with `{"mcpServers": {}}` |
| Invalid JSON | Restore from `.bak`, report error location |
| Server exists | Prompt: overwrite, rename, or skip |
| Missing env vars | Prompt user for values |
| Network timeout | Retry 3x with exponential backoff |
