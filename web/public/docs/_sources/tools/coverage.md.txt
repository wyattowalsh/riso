# Coverage.py

Coverage.py measures code coverage of Python programs. It monitors your program, noting which parts of the code have been executed.

## Features

- **Line coverage**: Track which lines were executed
- **Branch coverage**: Track which branches were taken
- **Reports**: HTML, XML, and terminal reports
- **Integration**: Works with pytest, unittest, etc.

## Usage in Riso

Coverage is integrated with pytest:

```bash
uv run coverage run -m pytest    # Run tests with coverage
uv run coverage report           # Terminal report
uv run coverage html             # HTML report
```

## Configuration

Riso typically uses a `coverage.cfg` or `pyproject.toml`:

```toml
[tool.coverage.run]
source = ["src"]
branch = true

[tool.coverage.report]
exclude_lines = ["pragma: no cover", "if TYPE_CHECKING:"]
```
