# Ruff

Ruff is an extremely fast Python linter and code formatter, written in Rust. It's 10-100x faster than existing linters and can replace Flake8, isort, pyupgrade, and many other tools.

## Features

- **Speed**: 10-100x faster than existing linters
- **Drop-in replacement**: Compatible with Flake8, isort, pyupgrade
- **Unified toolchain**: Linting and formatting in one tool
- **Auto-fix**: Automatically fix many issues

## Usage in Riso

Ruff is included in the `quality` dependency group:

```bash
uv run ruff check .        # Lint
uv run ruff format .       # Format
uv run ruff check --fix .  # Auto-fix
```

## Configuration

Riso configures Ruff in `pyproject.toml`:

```toml
[tool.ruff]
line-length = 88
target-version = "py311"

[tool.ruff.lint]
select = ["E", "F", "I", "UP"]
```
