# Copier

Copier is a library and command-line utility for rendering project templates. Riso itself is a Copier template.

## Features

- **Templates**: Jinja2-based project templates
- **Updates**: Keep projects in sync with template updates
- **Prompts**: Interactive configuration prompts
- **Tasks**: Post-generation automation hooks

## Usage

Generate a new Riso project:

```bash
uv run copier copy gh:wyattowalsh/riso my-project
```

Update an existing project:

```bash
uv run copier update
```

## Template Development

Riso templates use:
- `copier.yml` - Template configuration
- Jinja2 templating - `{{ variable }}`
- Pre/post hooks - Python scripts
