# Monorepo Tool Validator

The Monorepo Tool Validator ensures monorepo configurations are valid when using multi-package layouts.

## Purpose

- Validates workspace configurations
- Checks cross-package dependencies
- Ensures proper tooling for monorepos
- Validates pnpm workspace setup

## Location

```
template/hooks/validators/monorepo_tool_validator.py
```

## Validated Configurations

- uv workspaces for Python packages
- pnpm workspaces for Node packages
- Turborepo/Nx configurations (if enabled)

## Usage

Only invoked when `project_layout: monorepo` is selected.
