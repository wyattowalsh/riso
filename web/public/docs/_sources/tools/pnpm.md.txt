# pnpm

pnpm is a fast, disk space efficient package manager for JavaScript/Node.js. It uses a content-addressable store to save disk space.

## Features

- **Fast**: Up to 2x faster than npm
- **Disk efficient**: Content-addressable storage
- **Monorepo support**: Built-in workspace support
- **Strict**: Prevents phantom dependencies

## Usage in Riso

For projects with Node.js components:

```bash
pnpm install                # Install dependencies
pnpm add <package>          # Add a dependency
pnpm remove <package>       # Remove a dependency
pnpm run <script>           # Run a script
```

## Workspaces

Riso monorepos use pnpm workspaces:

```yaml
# pnpm-workspace.yaml
packages:
  - 'packages/*'
  - 'apps/*'
```

```bash
pnpm --filter <package> <command>
```
