# Tool Frontmatter Schema

This document defines the frontmatter schema for tool documentation pages.

## Required Fields

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Human-readable tool name |
| `type` | enum | Tool type (see below) |
| `category` | enum | Tool category (see below) |
| `description` | string | One-line description (max 120 chars) |
| `icon` | string | Iconify icon ID (e.g., `simple-icons:python`) |

## Optional Fields

| Field | Type | Description |
|-------|------|-------------|
| `version` | string | Version constraint (e.g., ">=1.0.0") |
| `homepage` | url | Official documentation URL |
| `repository` | url | Source code repository |
| `tags` | list[string] | Searchable keywords |
| `install_command` | string | Installation command |
| `requires` | list[string] | Dependencies |
| `provided_by` | enum | "template" or "external" |
| `popularity` | int | 0-100 ranking for sort (higher = more popular) |
| `brand_colors` | bool | Use official brand colors for icon (default: false) |

## Icon Selection Guide

Use [Iconify Icon Sets](https://icon-sets.iconify.design/) to find icons:

| Icon Set | Prefix | Best For |
|----------|--------|----------|
| Simple Icons | `simple-icons:` | Brand logos (Python, Docker, etc.) |
| Tabler Icons | `tabler:` | UI icons, actions, general purpose |
| Phosphor Bold | `ph:` | Bold, distinctive UI icons |
| Logos | `logos:` | Colored brand logos |
| Carbon | `carbon:` | Enterprise/IBM design style |
| Devicons | `devicon:` | Developer/tech logos |

## Valid Types

- `automation` - Build/task automation
- `linter` - Code linting
- `formatter` - Code formatting
- `type-checker` - Static type analysis
- `testing` - Test frameworks
- `package-manager` - Dependency management
- `container` - Containerization
- `ci` - Continuous integration
- `validator` - Input/config validation
- `mcp` - Model Context Protocol
- `framework` - Application frameworks
- `documentation` - Docs generation

## Valid Categories

- `riso-scripts` - Riso automation scripts
- `riso-validators` - Riso template validators
- `quality` - Code quality tools
- `packages` - Package managers
- `container-ci` - Container & CI tools
- `optional-modules` - Optional template modules
- `meta` - Documentation/schema files

## Example

```yaml
---
name: "Ruff"
type: "linter"
category: "quality"
description: "Extremely fast Python linter and formatter, written in Rust"
icon: "simple-icons:ruff"
popularity: 95
homepage: "https://docs.astral.sh/ruff/"
repository: "https://github.com/astral-sh/ruff"
tags: ["python", "linting", "formatting", "rust", "fast"]
install_command: "uv add ruff"
provided_by: "external"
brand_colors: true
---

# Ruff

Ruff is an extremely fast Python linter and code formatter...
```
