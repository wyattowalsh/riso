# FastAPI

FastAPI is a modern, fast (high-performance) Python web framework for building APIs. It's based on standard Python type hints and generates automatic OpenAPI documentation.

## Features

- **Fast**: One of the fastest Python frameworks
- **Type hints**: Automatic validation and serialization
- **OpenAPI**: Auto-generated interactive docs
- **Async**: Full async/await support

## Usage in Riso

Enable the `api_tracks: python` option to scaffold a FastAPI service:

```bash
# Run the development server
uv run uvicorn src.api.main:app --reload
```

## Generated Structure

```
src/
├── api/
│   ├── __init__.py
│   ├── main.py          # FastAPI app
│   ├── routers/         # Route modules
│   ├── schemas/         # Pydantic models
│   └── dependencies.py  # Dependency injection
```
