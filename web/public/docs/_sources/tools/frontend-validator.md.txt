# Frontend Validator

The Frontend Validator ensures valid combinations of frontend frameworks, component libraries, and build tools in Riso templates.

## Purpose

- Validates frontend framework selections (React, Vue, Svelte)
- Checks component library compatibility
- Ensures proper build tool configuration
- Prevents invalid combinations

## Location

```
template/hooks/validators/frontend_validator.py
```

## Validated Combinations

| Framework | Compatible Libraries |
|-----------|---------------------|
| React | shadcn, Radix, Chakra UI |
| Vue | Vuetify, PrimeVue |
| Svelte | Skeleton UI, Flowbite |

## Usage

Automatically invoked during template generation via pre-generation hooks.
