# Post-Generation Hook

The post-generation hook performs setup tasks after Copier finishes generating the template files.

## Purpose

- Initialize git repository
- Install dependencies
- Set up pre-commit hooks
- Display next-steps guidance

## Location

```
template/hooks/post_gen_project.py
```

## Execution

This hook runs automatically after template generation:

```bash
# Generation completes, then hook executes
copier copy . ./my-project
# -> Files generated
# -> post_gen_project.py runs
# -> Setup complete
```

## Tasks Performed

### Git Initialization

```python
# Initialize new git repository
subprocess.run(["git", "init"], check=True)
subprocess.run(["git", "add", "."], check=True)
subprocess.run(["git", "commit", "-m", "Initial commit from Riso template"], check=True)
```

### Dependency Installation

Based on project configuration:

| Configuration | Installation |
|---------------|--------------|
| Python backend | `uv sync --group dev` |
| Node frontend | `pnpm install` |
| Monorepo | Both, plus workspace setup |

### Pre-commit Setup

```python
subprocess.run(["uv", "run", "pre-commit", "install"], check=True)
subprocess.run(["uv", "run", "pre-commit", "install", "--hook-type", "commit-msg"], check=True)
```

### Cleanup

- Remove conditional template files that don't apply
- Clean up empty directories
- Fix file permissions

## Output

After completion, displays:

```
✅ Project generated successfully!

Next steps:
  cd my-project
  make quality     # Run linting and tests
  make docs        # Start documentation server
```
