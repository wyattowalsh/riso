# Generate Compatibility Matrix

This script parses `copier.yml` to extract prompts, dependencies, and incompatibilities, then generates comprehensive compatibility documentation.

## Purpose

- Extract prompt definitions from copier.yml
- Identify incompatible configuration combinations
- Generate Mermaid decision tree diagrams
- Document alternative recommendations and best practices

## Location

```
scripts/generate_compatibility_matrix.py
```

## Usage

### Generate Documentation

```bash
uv run python scripts/generate_compatibility_matrix.py
```

### Custom Output Path

```bash
uv run python scripts/generate_compatibility_matrix.py \
  --output docs/my-compatibility.md
```

### With Dependency Visualization

```bash
uv run python scripts/generate_compatibility_matrix.py \
  --visualize-deps
```

## Generated Content

### Compatibility Tables

- Backend language compatibility (frameworks, package managers, testing)
- Frontend framework compatibility (meta-frameworks, state management, components)

### Decision Trees (Mermaid)

```mermaid
graph TD
    A[Start: Backend Selection] --> B{Need API?}
    B -->|Yes| C{Performance Critical?}
    C -->|Yes| D[Rust + Axum]
    C -->|No| E[Python + FastAPI]
```

### Sections Generated

| Section | Content |
|---------|---------|
| Compatibility Matrix | Language/framework compatibility tables |
| Decision Trees | Mermaid diagrams for selection guidance |
| Alternative Recommendations | When to consider alternatives |
| Best Practices | Selection guidelines per category |
| Incompatible Combinations | Unsupported configurations |
| Prompt Dependencies | Which prompts depend on others |
