# Typer

Typer is a library for building CLI applications in Python, based on Python type hints. It's created by the same author as FastAPI.

## Features

- **Type hints**: Automatic argument parsing from type hints
- **Auto-complete**: Shell completion generation
- **Rich**: Beautiful terminal output with Rich
- **Testing**: Easy CLI testing with Click's test client

## Usage in Riso

Enable the `cli_module: enabled` option:

```bash
# Run your CLI
uv run mycli --help
uv run mycli command --option value
```

## Generated Structure

```
src/
├── cli/
│   ├── __init__.py
│   ├── main.py          # Typer app
│   └── commands/        # Subcommands
```

## Example

```python
import typer

app = typer.Typer()

@app.command()
def hello(name: str = "World"):
    typer.echo(f"Hello {name}!")

if __name__ == "__main__":
    app()
```
