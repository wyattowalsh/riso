# pytest

pytest is the most popular Python testing framework. It makes it easy to write simple and scalable tests.

## Features

- **Simple**: Easy to get started, powerful when needed
- **Fixtures**: Powerful fixture system for test setup
- **Plugins**: Rich ecosystem of plugins
- **Parametrize**: Easy test parameterization

## Usage in Riso

pytest is the default testing framework:

```bash
uv run pytest                    # Run all tests
uv run pytest tests/ -v          # Verbose output
uv run pytest -k "test_name"     # Run specific tests
uv run pytest --cov=src/         # With coverage
```

## Configuration

Riso configures pytest in `pyproject.toml`:

```toml
[tool.pytest.ini_options]
addopts = "-ra"
testpaths = ["tests"]
```
