# Pylint

Pylint is a comprehensive static code analyzer for Python. It checks for errors, enforces coding standards, and looks for code smells.

## Features

- **Comprehensive**: Checks errors, style, and code smells
- **Configurable**: Extensive configuration options
- **Plugins**: Extensible with custom plugins
- **Reports**: Detailed code quality reports

## Usage in Riso

Pylint is included for deep code analysis:

```bash
uv run pylint src/
uv run pylint --errors-only src/  # Only errors
```

## Configuration

Riso typically uses Pylint alongside Ruff for comprehensive coverage. Ruff handles fast linting while Pylint catches deeper issues.
